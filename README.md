## How to:
If you're having a bad day or just want a better day, do this:

* is ruby installed?
```
ruby -v
```
* clone repository
```
git clone git@github.com:cooljasonmelton/get-a-command-line-boost.git
```
* open directory and install dependencies 
```
bundle install
```
* run program
```
ruby run.rb
```
* answer question when prompted
